# NewLight

NewLight is a full stack project aimed to be showcase am ecommerce platform, it features shopping cart, search bar, and many other core features. 
It is done in Angular 19, TypeScript, NestJS and PostgreSQL and Cloudinary. Deployed initially on Render. You can see it live [here](https://newlight-store-frontend.vercel.app/).
The backend (NestJs) is [here](https://github.com/lejodev/new-light-backend)


# Tools used

## API
- TypeScript
- Angular 19
- Angular Material
- SASS

## Database
- PostgreSQL
